allez dans votre repo à vous

clickez sur NEW

  1. donnez un nom à votre repo
  2. vérifiez que public est coché.
    condition recquise pour pouvoir créer un lien partageable dans les prochaines étapes
  3. clickez sur Create repository

clickez sur upload an existing file

Ouvrez le fichier que vous avez clonez dans lequel vous avez intégré votre belle recette en html & css

A cette étape, il faut que le fichier index.html ne se trouve pas dans un Dossier comme sur l'image, le CSS et les Images peuvent se trouver dans un dossier, mais pensez à vérifier vos chemin relatif si jamais vous aviez votre index.html dans un dossier

Et glissez/déposez votre fichier HTML et votre fichier CSS dans la zone "Drag files here to add them to your repository"

Validez en cliquant sur "Commit changes"

Dirigez-vous dans les settings de ce Repo fraîchement créé

Descendez en scrollant jusqu'à la zone Github Pages

  1. cliquez sur "None"
  2. selectionnez "Main"
  3. puis cliquez sur "Save"

Pour Finir, patienter 15 à 30sec en rafraîchissant la page et en redescendant dans la zone "Github Pages" jusqu'à ce qu'un bandeau vert avec votre lien s'affiche !

Copiez/collez ce lien et faites-en ce que vous voulez !

Ce lien sera toujours dispo dans les Settings du Repo Pancakes dans le bas de page Github Pages

Bon code !
